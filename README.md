Bütçe Takip Uygulaması

JavaScript pratiği yapmak için geliştirdiğim basit bir bütçe takip uygulaması.

Özellikler:

Gelir ve gider ekleme
Kategorilere göre ayırma
Tarih bazlı kayıt
Filtreleme (hepsi/gelir/gider)
LocalStorage ile veri saklama

Kullanılan Teknolojiler:
HTML5
CSS3
Vanilla JavaScript
LocalStorage API

Bu Projeyi Yaparken Öğrendiklerim:

LocalStorage kullanımını öğrendim
DOM manipülasyonu pratiği yaptım
JavaScript array metodlarıyla (filter, sort) çalıştım
Responsive tasarım uyguladım
